♦ Pre-owned Car Market ♦

Design Study

This project is aimed at creating interactive visualization to understand the relation between price of pre-owned cars and various features such as car brand, condition, mileage, location etc.

Dataset

We are using an open source data set of used cars from Kaggle ( link ) which consists of 400K records and 26 features.

After cleaning and pre-processing data using python, our final data contains:

Number of features: 11

Features: Car id, City, State, Manufacturer, Model, Condition, Title status, Odometer, Size, Type of car, Posted Date.

Number of records: 240K
